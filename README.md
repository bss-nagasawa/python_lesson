# Project Title

このプロジェクトは、地理的な位置情報を管理するためのアプリケーションです。ユーザーは地点を登録し、それらの位置情報を基に様々な分析を行うことができます。

## Installation

このプロジェクトを実行するには、Python 3.8 以上が必要です。また、SQLite をデータベースとして使用しています。

1. 依存関係のインストール:
   ```bash
   pip install -r requirements.txt
   ```
